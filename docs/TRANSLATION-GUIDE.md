# Translation Guide

## Text Domain
**palmo-trans-de** (theme)
**palmo-calculator** (plugin)

## Workflow
1. Extract strings: `wp i18n make-pot`
2. Translate .pot to .po
3. Compile .mo files
4. Place in languages/

## German Translation
Primary language: German (de_DE)
Fallback: English (en_US)

## Tools
- Poedit
- Loco Translate (WordPress plugin)
- WP-CLI i18n commands
